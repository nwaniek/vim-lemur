*vim-lemur.txt*    Syntax highlighting for Lemur

INTRODUCTION
    This plugin provides Vim syntax highlighting for the Lemur markup language.

INSTALLATION
    Use a plugin manager like vim-plug:
    
        Plug 'nwaniek/vim-lemur'

    Or manually:
    
        mkdir -p ~/.vim/{syntax,ftdetect,ftplugin,indent,doc}
        cp syntax/lemur.vim ~/.vim/syntax/
        cp ftdetect/lemur.vim ~/.vim/ftdetect/
        cp doc/lemur.txt ~/.vim/doc/
